# Example Descriptions

## Overview

This document provides examples of scripts that demonstrate various functionalities of the market simulation framework. Each script is designed to showcase a specific feature or use case.

## Example Scripts
- **`run_simulation.py`**: Runs a complete market simulation.
- **`run_nlp_config_generator.py`**: Generates configurations using natural language.
- **`run_order_flow_generator.py`**: Generates order flows in parallel.
- **`run_impact_evaluator.py`**: Evaluates market impact.
- **`run_dynamic_stop_loss.py`**: Tests dynamic stop-loss strategies.
